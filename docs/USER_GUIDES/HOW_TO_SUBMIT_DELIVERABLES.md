## Overview

### What are Deliverables?

Deliverables are the content pieces you need to create and submit for a campaign. These can include:

- Instagram posts
- Instagram reels
- YouTube videos
- Stories
- Blog posts
- And more...

### When to Submit

- After being **SELECTED** for a campaign
- Before the **deadline** specified in the campaign
- After reviewing all **requirements** carefully

---

## Viewing Deliverables

### Step 1: Access Deliverables

1. Go to **Applications** tab
2. Find your **SELECTED** application
3. Tap on the application
4. Tap **"View Deliverables"**

### Step 2: Review Requirements

For each deliverable, you'll see:

- **Type**: Post, Reel, Video, etc.
- **Title**: Deliverable name
- **Description**: What needs to be created
- **Due Date**: Submission deadline
- **Is Mandatory**: Required or optional
- **Brand Guidelines**: Specific requirements

### Important Information

- ✅ **Deadline**: Submit before the due date
- ✅ **Format**: Check required file formats
- ✅ **Size**: Check file size limits
- ✅ **Content Requirements**: Follow brand guidelines exactly
- ✅ **Hashtags**: Use required hashtags
- ✅ **Mentions**: Tag the brand account

## Preparing Your Content

### Step 1: Review Brand Guidelines

Before creating content:

1. Read all brand guidelines carefully
2. Check required:
   - Hashtags
   - Mentions/tags
   - Caption style
   - Visual style
   - Product placement
3. Save guidelines for reference

### Step 2: Plan Your Content

1. **Brainstorm Ideas**
   - What story will you tell?
   - How will you showcase the product?
   - What makes it unique?

2. **Create Shot List**
   - What shots do you need?
   - What locations?
   - What props?

3. **Schedule Time**
   - Plan enough time for creation
   - Leave buffer for revisions
   - Don't wait until the last minute

### Step 3: Create Content

1. **Follow Guidelines**
   - Use required hashtags
   - Tag brand account
   - Follow visual style
   - Include required elements

2. **Quality Standards**
   - High-resolution images/videos
   - Good lighting
   - Clear audio (for videos)
   - Professional editing

3. **Be Authentic**
   - Show genuine experience
   - Be creative
   - Stay true to your style

## Uploading Deliverables

### Step 1: Select Deliverable

1. Go to **Deliverables** tab
2. Find the deliverable you want to submit
3. Tap on it
4. Tap **"Submit"**

### Step 2: Upload Files

1. **Choose File Type**:
   - Photo
   - Video
   - Document

2. **Select File**:
   - Tap "Choose File"
   - Select from gallery or camera
   - For videos, can record directly

3. **File Requirements**:
   - **Photos**: JPG, PNG (max 10MB)
   - **Videos**: MP4, MOV (max 100MB)
   - **Documents**: PDF (max 5MB)

### Step 3: Add Details

1. **Caption** (if required):
   - Write your caption
   - Include required hashtags
   - Tag brand account
   - Check character limit

2. **Additional Notes** (optional):
   - Any special notes for the brand
   - Questions or clarifications

### Step 4: Review and Submit

1. **Review**:
   - Check file is correct
   - Verify caption/hashtags
   - Ensure all requirements met

2. **Submit**:
   - Tap **"Submit Deliverable"**
   - Wait for upload to complete
   - Confirmation message appears

### Step 5: Confirmation

After submitting:
- ✅ Status changes to "SUBMITTED"
- ✅ You'll receive a confirmation
- ✅ Brand will review your submission
- ✅ You can track status

## Submission History

### View History

1. Go to **Deliverables** tab
2. Tap on a deliverable
3. Tap **"View History"**
4. See all submission attempts

### History Shows

- **Submission Date**: When you submitted
- **Status**: SUBMITTED, APPROVED, REJECTED
- **Files**: What you uploaded
- **Feedback**: Brand comments (if any)
- **Revision Number**: If resubmitted

## Revision Requests

### What is a Revision?

If your submission doesn't meet requirements, the brand may request revisions. You'll need to:

1. Review feedback
2. Make changes
3. Resubmit

### Step 1: Receive Feedback

1. You'll receive a notification
2. Go to **Deliverables** tab
3. Tap on the deliverable
4. View brand feedback

### Step 2: Make Changes

1. **Read Feedback Carefully**:
   - What needs to change?
   - Why was it rejected?
   - What are the specific requirements?

2. **Update Content**:
   - Make requested changes
   - Follow feedback exactly
   - Don't make unrelated changes

3. **Re-upload**:
   - Follow same upload process
   - New submission replaces old one
   - Revision number increments

### Step 3: Resubmit

1. Upload updated files
2. Add any new notes
3. Submit again
4. Wait for review

### Revision Tips

- ✅ Address all feedback points
- ✅ Don't take feedback personally
- ✅ Ask questions if unclear
- ✅ Resubmit promptly

## Common Questions

### What if I miss the deadline?

Contact the brand immediately. Some brands may allow late submissions, but it's not guaranteed.

### Can I submit multiple versions?

No, each submission replaces the previous one. Make sure your submission is final before submitting.

### What if my file is too large?

- Compress images/videos
- Use recommended formats
- Contact support if needed

### How long does review take?

Usually 2-5 business days, but can vary. You'll receive a notification when reviewed.

### What if I need to change something after submitting?

Contact the brand through messaging. They may allow you to resubmit.
